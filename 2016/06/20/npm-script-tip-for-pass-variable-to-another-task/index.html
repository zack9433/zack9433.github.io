要作 e2e 測試所以，先起 phantomjs 之後再起 protractor。

以上會發現當 protractor 跑完測試後，phantomjs 並沒有停止執行，所以必須在執行完 protractor 後要停止 phantomjs。作法大致上是紀錄 phantomjs pid 然後在 protractor 執行完後 kill phantomjs pid。
